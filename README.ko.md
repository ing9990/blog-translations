# 📚 기술 블로그 번역 저장소

이 저장소는 다양한 기술 블로그를 **한국어로 번역하여 공유하는 공간**입니다.  
프로그래밍, 인프라, 머신러닝, 프론트엔드, 블록체인 등 다양한 기술 주제를 다룹니다.

누구든 자유롭게 번역글을 추가할 수 있으며, 새로운 기여는 언제든지 환영합니다! 🙌

---

## 디렉토리 설명

- `/ko`: **한국 이외**의 기술 블로그를 **한국어로 번역**하는 디렉토리입니다.
- `/en`: **미국 이외**의 기술 블로그를 **영어로 번역**하는 디렉토리입니다.
